# React Native Basic Course 📱

This repository contains my foundational projects and exercises from learning React Native. Each folder explores different topics, such as navigation, styling, networking, and dynamic UI components.

## 🗂️ Projects Included

- **HelloWorld**: Initial React Native setup and basic components.
- **HelloStyling**: Deep dive into Styling in React Native.
- **Hello_DynamicUI**: Creating dynamic and responsive user interfaces.
- **PokemonCards**: A fun project involving rendering lists of Pokemon cards.
- **NetWorking**: Handling API requests and networking in React Native.
- **Navigation Modules**:
  - `DrawerNavigation`: Implementing a drawer menu.
  - `TabNavigation`: Implementing bottom tab navigation.
  - `StackNavigation2`: Implementing stack-based navigation.
  - `RNnavigation`: General React Navigation concepts.
  - `ExpoNavigation`: Using Expo Router for file-based routing.

## 🚀 Getting Started

Navigate to any project directory, install dependencies, and run:

```bash
npm install
npm start
```
